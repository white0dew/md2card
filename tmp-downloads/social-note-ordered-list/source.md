# 四步搭建多智能体协作团队

如果你已经成功运行过 Hermes，只需四步就能建立一套体系化的多智能体协作团队：

1. 克隆配置档案
在终端执行命令：`hermes profile create "xxx" --clone`
新档案会继承已调试好的基础参数，同时拥有独立的记忆与对话会话，互不干扰。

2. 为每个档案编写 SOUL.md
明确各智能体的角色定位与职责边界。SOUL.md 用于定义该智能体的核心身份、专长领域以及任务禁区。

3. 在项目根目录配置 AGENTS.md
AGENTS.md 统一存放项目架构、协作规范与任务推进进度，为所有智能体提供一致的任务背景信息。

4. 完成配置后分别调用，实现分工协作
让不同智能体处理不同子任务，避免任务串行处理带来的效率损失。
